# Books Management App

A very simple Spring Boot practice project made for learning purposes.

## Features
- View all books ("/books") GET
- Get a book by id ("/books/book/{id}") GET
- Add a book ("/books") POST
- Add books in bulk ("/books/bulk") POST
- Update a book ("/books/update/{id}") PUT
- Delete a book ("/books/delete/{id}") DELETE 

## Tech Stack
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL Database

## Note
This is a first practice project and uses simple in-memory (List) storage.
